backend는 사용자가 볼 수 없는 웹 사이트 또는 s/w의 모둔 부분을 의미한다.
UI를 참조하는 frontend와 대조된다.

보통 사용되는 backend의 프로세스는 다음과 같다.
1. 웹 페이지의 request 처리
2. script 실행을 통한 html 생성
3. DB 관련 처리
4. 데이터 암호화 및 해독
5. 파일 업로드 및 다운로드 처리
6. javascript를 통한 사용자 입력 처리

1 ~ 5번까지는 웹 서버측에서 실행되는 프로세스 항목이고,
6번은 클라이언트에서 실행되는 프로세스 항목이다. 그래서 6번은 때때로 frontend로 취급받기도 한다.

backend는 frontend와 같이 동작해야 완전한 상태를 이루기 때문에 결론은 둘 다 중요하다.
